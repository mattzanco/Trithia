# Trithia - 2D RPG Game

A 2D RPG adventure game built with Godot Engine.

## Project Structure

```
Trithia/
├── scenes/           # Game scenes
│   ├── main.tscn    # Main entry scene
│   ├── player/      # Player character scenes
│   ├── enemies/     # Enemy character scenes
│   ├── levels/      # Level/map scenes
│   └── ui/          # User interface scenes
├── scripts/         # GDScript files
│   ├── main.gd      # Main game logic
│   └── player.gd    # Player controller
├── assets/          # Game assets
│   ├── sprites/     # Sprite images
│   ├── audio/       # Sound effects and music
│   ├── fonts/       # Font files
│   └── tilesets/    # Tileset images
├── project.godot    # Godot project configuration
└── icon.svg         # Project icon
```

## Getting Started

1. Install [Godot Engine 4.3+](https://godotengine.org/download)
2. Open the project in Godot by selecting the `project.godot` file
3. Press F5 to run the project

## Development

- The main scene is located at `scenes/main.tscn`
- Player controller script is at `scripts/player.gd`
- Add your sprites to `assets/sprites/`
- Add tilesets to `assets/tilesets/`
- Create new levels in `scenes/levels/`
